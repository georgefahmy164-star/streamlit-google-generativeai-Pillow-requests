# 💀 Ultimate AI God Mode

🔥 AI Web App شبيه بـ ChatGPT مع قدرات قراءة ملفات PDF، ذاكرة محادثة، وRAG للردود الذكية.

## 🚀 المميزات
- 🤖 AI Chat شبه ChatGPT
- 📄 رفع PDF + استخراج المعرفة
- 🧠 Memory & Conversation History
- 🔎 RAG للردود الذكية
- 🌐 Web Interface تفاعلية
- 🐳 Docker جاهز للنشر عالميًا
